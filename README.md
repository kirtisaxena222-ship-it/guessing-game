# 🎯 Number Guessing Game (C++)

A simple command-line Number Guessing Game written in C++.

## 🚀 Features
- Random number generation
- Difficulty levels (Easy / Medium / Hard)
- Hint system (Too High / Too Low)
- Attempt counter
- Replay option

## 🛠 Technologies Used
- C++
- Standard Library (cstdlib, ctime)

## ▶ How to Run

1. Compile:
   g++ main.cpp -o game

2. Run:
   ./game
